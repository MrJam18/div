Для запуска проекта:
<ol>
<li>composer install</li>
<li>yii migrate</li>
<li>yii migrate --migrationPath=@yii/rbac/migrations</li>
<li>yii rbac/init</li>
</ol>
<p>
    Все сделано в соответствии с заданием.
</p>
